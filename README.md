# Simple example websocket using nodejs and go


## Run Nodejs as Server
 - installing node packages
 ``` bash
npm install
 ```
 - run server
 ``` bash 
 nodemon main.js
 ``` 

---
## Run Golang as Server
 - installing go packages
 ``` bash
go mod tidy
 ```
 - run client
 ``` bash 
 go run main.go
 ```
